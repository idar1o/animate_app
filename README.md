# animate_app

Описание

Этот репозиторий посвящён изучению и экспериментам с шейдерами в мобильной разработке. Здесь я реализую современные визуальные эффекты, такие как liquid glass (glassmorphism), blur, градиенты и кастомный рендеринг UI.

Цель проекта — глубже понять работу шейдеров на практике и научиться применять их для создания продвинутых и производительных пользовательских интерфейсов в мобильных приложениях.

⚙️ Возможности
Эффекты liquid glass / glassmorphism
Кастомные шейдеры для UI-компонентов
Эксперименты с blur, градиентами и освещением
Подходы к оптимизации рендеринга
